# Bot Sintraport

Bot para WhatsApp que gerencia reservas de:
- Chácara
- Transporte médico

## Funcionalidades
- Verificação de disponibilidade
- Reserva de datas
- Confirmação automática
- Registro em banco de dados

## Tecnologias
- Node.js
- Express
- SQLite
- Twilio

## Configuração
Necessário criar arquivo .env com:
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN

## Instalação
```bash
npm install
npm start
